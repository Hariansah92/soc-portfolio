# 🛑 Web Recon & Exploit Attempt Detection

## 📌 Scenario
Detected suspicious reconnaissance and exploitation attempts targeting a public-facing web application.

## 🚨 Detection
Alert generated from WAF & SIEM indicating suspicious request to /phpinfo.php

## 🔍 Analysis
- Repeated request from attacker IP
- Malformed query string detected

## ⚡ Response
- Blocked attacker IP
- Enabled WAF protection

## 📊 Outcome
- Attack contained successfully
